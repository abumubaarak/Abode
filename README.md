<h1 align="center">Abode for Tenants</h1> 
<p>Abode is a home rental platform that allows landlords to publish listings, and tenants can find and secure their next home with ease.</p> 
</p>

<img  src="./brand/stepper.png"/>

## Features ✨
- Browse Listings.
- Apply for listing.
- View Listings.
- Instant Messaging.
- Payment.
- Search.
- Wishlists.
